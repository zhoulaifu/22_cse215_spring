* Exam Problem 2, Midterm-1 2021

  Check the logical equivalence of ((p ∧ q) → r) and ((p → r) ∨ (q → r))

** try
   
  Approach 1 : truth table

  Approach 2: logical equivalence

  (p /\ q) -> r
= ~(p /\ q) \/ r   // A->B is the same as ~A \/ B
= ~p \/ ~q \/ r

  (p -> r) \/ (q -> r)
= (~p \/ r) \/ (~q \/ r) // A->B law
= ~p \/ (r \/ ~q) \/ r  // associative law
= ~p \/ (~q \/ r) \/ r // commutative law A \/ B = B\/ A\
= ~p \/ ~q \/ r \/ r
= ~p \/ ~q \/ (r \/ r)
= ~p \/ ~q \/ r         //idempotent A \/ A = A

* Exam Problem 2, Final 2020-1  
Suppose p and q are propositional statements. Prove that p and q are logically equivalent if and only if p ↔ q is a tautology.

Category: tautology; logic equivalence

** try

   p <-> q is tautology means: for any p, q, (p->q) /\ (q->p)

   p and q are logically equivalent means (p /\ q)\/(~p /\ ~q)

   First approach: truth table

  | p | q | expression-A | expression-B |
  |---+---+--------------+--------------|
  |   |   |              |              |

   Second approach: logical equiv.

   (p -> q) /\ (q -> p)
 = (~p \/ q) /\ (~q \/ p)  // A -> B law
 = ((~p \/ q) /\ ~q ) \/ ((~p \/ q) /\ p)
 = (~p /\ ~q) \/ (q /\ p)
